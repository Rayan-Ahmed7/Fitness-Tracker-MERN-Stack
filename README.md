# 🏋️ Fitness Tracker MERN Stack

A modern **Full Stack Fitness Tracker** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**. The application helps users manage workouts, track nutrition, monitor fitness progress, and securely manage their accounts.

---

## 🚀 Features

- 🔐 User Authentication (Register & Login)
- 🛡️ JWT Protected Routes
- 💪 Workout Management
- 🥗 Nutrition Tracking
- 📊 Fitness Progress Analytics
- 👤 User Profile Management
- 📱 Responsive User Interface
- 🌐 RESTful API
- 💾 MongoDB Database Integration

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- Axios
- CSS

### Backend
- Node.js
- Express.js
- JWT Authentication
- bcrypt.js

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Rayan-Ahmed7/Fitness-Tracker-MERN-Stack.git
```

### Navigate to the Project

```bash
cd Fitness-Tracker-MERN-Stack
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd backend
npm start
```

### Start Frontend

```bash
cd frontend
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file inside the `backend` folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
